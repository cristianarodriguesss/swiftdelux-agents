      Ligar Instagram automaticamente

      Para teres métricas automáticas precisas de ligar o Instagram Graph API. Aqui estão os passos:

      Passo 1 — Conta Profissional
      A conta do Instagram tem de ser Profissional (Business ou Creator). Settings → Account → Switch to Professional Account.
      Passo 2 — Criar App no Meta
      Vai a developers.facebook.com → Criar App → tipo "Business" → adiciona o produto "Instagram Graph API".
      Passo 3 — Obter Access Token
      No Graph API Explorer, gera um token com permissões: instagram_basic, instagram_manage_insights, pages_read_engagement.
      Passo 4 — Adicionar ao GitHub
      Adiciona o token como secret INSTAGRAM_TOKEN e o ID da conta como INSTAGRAM_ACCOUNT_ID.

      Este processo demora ~30-45 minutos. Posso guiar-te passo a passo quando quiseres.
